# ShipMates Legal Documents

This directory contains the public legal documents for the ShipMates app, hosted via GitHub Pages.

## Live URLs (once deployed):
- Privacy Policy: `https://[username].github.io/shipmates/privacy-policy.html`
- Terms of Service: `https://[username].github.io/shipmates/terms-of-service.html`
- Community Guidelines: `https://[username].github.io/shipmates/community-guidelines.html`

## Files:
- `index.html` - Landing page with links to all documents
- `privacy-policy.html` - Full privacy policy
- `terms-of-service.html` - Terms of service
- `community-guidelines.html` - Community guidelines
- `.nojekyll` - Tells GitHub Pages not to use Jekyll processing
